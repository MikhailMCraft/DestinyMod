The Guardian Games are on! :party:
Added the Podium, purchasable from Zavala for the duration of the Games
Added Laurels, which drop from enemies during the Games with a 1/25 chance
Added Titan, Warlock, and Hunter-themed decor

Completely overhauled class selection
Classes are now chosen on character creation and cannot currently be changed afterwards
Classes are shown on the player's character bar on the character select screen
Removed class marks
Fixed flickering that would occur when using No Land Beyond
Made Rat King an automatic gun (from semi-automatic)
Lowered Sweet Business' chance to consume ammo to 25% (from 10%)
Fixed an issue which would allow players to place class-restricted armor in armor slots when they were not allowed to
Fixed an issue where Borealis would not sync projectile types in multiplayer
Fixed an issue where Ticuu's Divination's firing sounds would only play to the owner of the weapon
Added a lot more Polish and Spanish translations
Titan Prodigal set now has tooltips if the player is not a Titan and has the "strict-class" config enabled
Fixed a typo in the Prodigal Helm's description
Fixed an issue which would allow users to place a class-restricted item in their vanity slot and then right-click to equip it
Decreased the volume of Divinity and all fusion rifles
